2nd Year Project

For my 2nd Year Project my group created a live website for aviation fans to get news and to get updates on various airplanes using JavaScript, HTML and CSS. The main functionality of our project is that we wanted to create a system where a user subscribes to our subscription system and could receive updates when subscribed. We added an extra feature such as plane tracking so users that users can go the website and to see what planes there are currently in the air and where they are if they had more time designated.
